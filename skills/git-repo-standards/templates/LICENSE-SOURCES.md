# License Templates

Do not store full license text in templates. Fetch from authoritative sources:

## Apache-2.0 (Libraries, CLIs, Tools)
- Source: https://www.apache.org/licenses/LICENSE-2.0.txt
- SPDX: https://spdx.org/licenses/Apache-2.0.html

## AGPL-3.0 (Services, APIs, Platforms)
- Source: https://www.gnu.org/licenses/agpl-3.0.txt
- SPDX: https://spdx.org/licenses/AGPL-3.0-only.html

## Usage

When generating a new repo:

1. Determine license type based on project classification
2. Fetch license text from authoritative source
3. Add copyright header with current year and "fyrsmithlabs"
4. Save as `LICENSE` (no extension) in repo root
